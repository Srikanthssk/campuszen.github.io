✦ Core Competencies

What You Will
Master

Great design is equal parts empathy and craft. This program teaches you both — from understanding users deeply to building interfaces that are as beautiful as they are functional, using the tools top studios rely on daily.

Understand Your Users

Master user research, interviews, and personas to build deep empathy and uncover true user needs.

Craft Beautiful Interfaces

Learn visual design principles, color theory, typography, and create pixel-perfect UIs with design systems.

Build Seamless Interactions

Go from wireframes to high-fidelity, interactive prototypes that feel exactly like real products.

Validate Your Designs

Conduct usability testing, analyse feedback, and iterate on your designs based on real user data.

✦ Course Structure

Your Journey Through
the Design Process

Six modules that mirror exactly how professional designers work — from discovering user problems to delivering polished, handoff-ready designs and a portfolio case study.

Understand the core principles of user-centered design, learn the Design Thinking framework, and define user problems with precision.

Design Thinking UX Principles Problem Framing Empathy Mapping

Conduct user interviews, create personas and journey maps, and build a solid information architecture that guides every design decision.

User Interviews Personas Journey Mapping Information Architecture

Translate your research into low-fidelity wireframes and build interactive, clickable prototypes to test user flows before any visual design begins.

Lo-Fi Wireframes User Flows Interactive Prototypes Figma

Master color theory, typography, and layout principles. Learn to create design systems and apply branding consistently across your interfaces.

Color Theory Typography Design Systems Hi-Fi UI

Plan and conduct usability tests, analyse feedback for iteration, and prepare your final designs for seamless developer handoff using Zeplin.

Usability Testing A/B Testing Iteration Zeplin Handoff

Apply the entire design process to a real-world project, culminating in a portfolio-ready case study that showcases your full design thinking and execution.

End-to-End Project Case Study Writing Portfolio Review Interview Prep
